# 用户使用手册

## 简介

A股行情监控系统是一款专为A股投资者设计的实时行情监控工具，可以帮助用户快速了解自选股的实时价格变动和市场整体走势。

## 系统要求

- Windows 7及以上操作系统
- Python 3.7+（源码运行）
- 至少100MB可用磁盘空间

## 安装与运行

### 方法一：使用打包版本（推荐）

1. 访问项目的 [GitHub Releases](https://github.com/taoste/stock/releases) 页面
2. 下载最新版本的 `stock_monitor.zip` 文件
3. 解压缩到任意目录
4. 双击运行 `stock_monitor.exe`

### 方法二：源码运行

1. 确保已安装Python 3.7+
2. 克隆或下载项目源码
3. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
4. 运行程序：
   ```bash
   python stock_monitor/main.py
   ```

## 界面介绍

### 主界面

主界面是一个悬浮在桌面上的透明窗口，显示用户的自选股实时行情：

- **股票名称**：显示股票的简称
- **当前价格**：显示股票的最新成交价格
- **涨跌幅**：显示相对于昨日收盘价的涨跌幅度，用颜色区分：
  - 红色：上涨
  - 绿色：下跌
  - 灰色：平盘
- **封单信息**：对于涨停或跌停的股票，会显示封单手数

底部的状态条以可视化方式展示整个市场的涨跌情况。

### 右键菜单

在主界面上点击鼠标右键，可以打开菜单：

- **设置**：打开设置对话框
- **退出**：退出程序

## 功能使用

### 自选股管理

1. 在主界面右键选择"设置"
2. 在设置对话框中可以通过以下方式添加自选股：
   - 在搜索框中输入股票代码或名称
   - 从搜索结果中双击股票添加到自选股列表
   - 按回车键添加第一个搜索结果
3. 在自选股列表中可以：
   - 通过拖拽调整股票顺序
   - 选择股票后点击"删除选中"按钮删除
4. 点击"确定"保存设置

### 刷新频率设置

在设置对话框中，可以选择不同的刷新频率：
- 2秒：高频刷新，适合短线交易者
- 5秒：标准刷新频率
- 10秒：平衡刷新频率和资源消耗
- 30秒：低频刷新，节省网络流量

### 开机启动

在设置对话框中勾选"开机启动"选项，可以使程序在系统启动时自动运行。

### 版本更新

在设置对话框中点击"检查更新"按钮可以手动检查是否有新版本。如果有新版本，程序会自动下载并提示重启以应用更新。

## 常见问题

### 为什么有时数据显示为"--"？

这通常是因为网络连接问题或者股票停牌导致无法获取实时数据。可以稍后再试。

### 如何添加ST股票？

在搜索框中输入ST股票的完整代码（如`sz002XXX`）即可搜索并添加。

### 程序运行时占用内存较大怎么办？

这是正常现象，程序会在后台缓存一些数据以提高响应速度。如果确实影响系统性能，可以适当降低刷新频率。
系统已实现动态缓存优化，会根据可用系统资源自动调整缓存大小以平衡性能和资源消耗。

### 如何卸载程序？

如果是使用打包版本，直接删除解压的文件夹即可。程序不会在系统中留下其他痕迹。

## 故障排除

### 程序无法启动

1. 检查系统是否满足最低要求
2. 确认所有文件都在同一目录下
3. 检查杀毒软件是否阻止了程序运行

### 数据不更新

1. 检查网络连接是否正常
2. 确认当前处于交易时间内（周一至周五 9:30-11:30, 13:00-15:00）
3. 尝试重启程序

### 设置无法保存

1. 确认程序有写入当前目录的权限
2. 检查磁盘空间是否充足
