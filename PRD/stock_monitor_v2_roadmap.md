# Stock Monitor V2 渐进式重构路线图 (Roadmap)

**文档状态**: 待执行
**基本原则**: 采用“渐进式重构”，每一阶段都保证可独立运行、不出大面积流血，绝不为了“完美”丢弃已经生效跑通的核心代码。

---

## Phase 1: 扫清体验障碍 (即刻修补)

### 🥇 核心目标：强修“桌面置顶失效”及前台底层 Bug
*   **动作 1 (Windows Native Hook)**：修改 UI 类（即 `ui/main_window.py` 等的设置层面）。引入 `ctypes.windll.user32.SetWindowPos` 的超级内核锁，取代或加强被 Windows 软抢占焦点打掉的 `Qt.WindowStaysOnTopHint`。
*   **动作 2 (排查渲染挂起)**：分析现有的数据轮询方法，确保表格颜色刷新阶段没有任何硬阻塞逻辑（`sleep` 等）。
*   **预期产出**：一套永远贴在屏幕右上角的坚固监测看板，不再因切屏遗漏任何大单。
*   **工期预估**：极短（1 次 Sprint，即刻可落地）。

---

## Phase 2: 消除数据代沟 (大一统底层)

### 🔁 核心目标：完成到 `mootdx` 的纯血切换
*   **动作 1 (解耦与剔除)**：在 `core/stock_data_fetcher.py` 和整个项目里，彻底下线 `easyquotation` 所有的调用和轮询线程。
*   **动作 2 (平铺切轨)**：将基础行情字段的获取指向 `mootdx.quotes`。利用该接口自带的多标的并发能力（一次可传 80 支股代码），重置所有涨跌幅刷新管线。
*   **预期产出**：实现涨跌幅和逐笔数据的真正毫秒级时间戳同步，杜绝“一锅炒两口饭”的数据打架隐患。
*   **工期预估**：短（1 次 Sprint，只需修改底层获取与一层映射，不改前端）。

---

## Phase 3: AI 大脑与超级预警觉醒 (量化基石)

### 🧠 核心目标：引入 `pandas-ta`，接入微信总线
*   **动作 1 (量化信号池搭建)**：新建 `analysis` 模块。订阅并基于最近一次全量 `mootdx.bars` 运行 MACD 底背离、OBV、VWAP 突破、甚至超级趋势阻力的算力检查。
*   **动作 2 (分离核心解耦 MVP)**：UI 层脱离大单数据查询的苦海，“发心率广播”，独立出真正的纯异步微服务（`Engine` / `Worker`）。它自己算出背离与破位。
*   **动作 3 (跨端通信接入)**：实现独立于桌面的推流层，建立 `ServerChan / PushPlus` HTTP 通知通道，将触发的异动通过 JSON 组合好发入您的私人微信对话框。
*   **预期产出**：系统首次脱离显示器独立生效，即便是关着界面/外卖买咖啡，微信也能实时弹窗“XXX 发生了 5分钟级别的技术背离”。
*   **工期预估**：中长（核心战役，重构调度器和运算引擎）。

---

## Phase 4: 长线战略与终极收官 (大后方闭环)

### 📊 核心目标：每天 15:05 智能复盘及本地筹码微账本
*   **动作 1 (数据库选型与入列)**：使用自带无需部署内存负担的 `TinyDB` 或 Python 原生 `.json` 系统建立数据仓位。
*   **动作 2 (锚点持久化)**：当您的鼠标点击“加入某个股票到自选”的瞬间，抓拍当前的最新日线价格，以及最重要的 `mootdx.finance` 取得的股东人数，固化为历史快照。
*   **动作 3 (筹码低频预警)**：建立 9:25 分启动脚本，执行最新财报与那一张“快照”的增除幅度计算，变动超过 ±5%，警示筹码集中/发散异象。
*   **动作 4 (APScheduler 收盘写稿人)**：加入 `Daily_15:05_Reporter`。扫描当天全网大单、顶底背离触发股池、长线主力动作，汇总为精美 Markdown 微信每日复盘邮件。
*   **预期产出**：一套完美的量化策略个人工作站，实现从“盘前提醒、盘中预警信号、盘后自动复盘”全生命周期管线。
*   **工期预估**：中（逻辑清晰，功能模块化，逐一添加不影响主线程）。
