# 股票数据技术指标导出指南

本指南介绍如何使用项目中提供的统一导出脚本 **[export_stocks_to_excel.py](file:///d:/code/stock/scripts/reporting/export_stocks_to_excel.py)**，抓取自选股历史指标或全量 A 股行情数据并导出为排版精美的 Excel 报表。

---

## 一、 功能概述

该导出工具整合了多项股票导出功能，并支持通过命令行参数（开关）进行灵活控制：

1. **读取自选股**：自动加载当前用户的自选股列表（从用户本机的 `config.json` 配置文件中的 `user_stocks` 键读取）。
2. **高速抓取历史 K 线**：使用底层 TCP 协议的 `mootdx`（通达信）行情服务批量抓取指定交易日（默认 350 个交易日）的日线历史数据。
3. **计算量化指标**：
   - **K线均线**：MA5、MA10、MA20、MA60
   - **布林带 (BOLL)**：上轨、中轨、下轨
   - **平滑异同移动平均线 (MACD)**：DIFF、DEA、BAR (红绿柱)
   - **相对强弱指标 (RSI)**：RSI_14
   - **量能指标**：成交量、成交额、成交量 MA5、成交量 MA20
4. **全量 A 股实时行情**：支持多源自动重试机制（优先 Eastmoney，自动回退至 Sina 备份接口）。
5. **精美 Excel 样式渲染与红涨绿跌高亮**：
   - 自动调整列宽，中文字符加宽自适应。
   - 对涨跌幅和 MACD_BAR 进行红涨绿跌（红字红底，绿字绿底）高亮。
   - 自动对齐：文本居中，数值右对齐。

## 二、 模式开关说明

脚本提供了命令行选项，您可以按需切换：

| 命令行开关 | 说明 | 导出内容 |
| :--- | :--- | :--- |
| `-m watchlist` 或 `--mode watchlist` | **自选股模式（默认）** | 自选股的技术面概览、自选股合并历史明细、每只自选股的独立历史数据页。 |
| `-m all` 或 `--mode all` | **全量实时模式** | 全量 5000+ A股最新的实时行情快照 Sheet。 |
| `-m both` 或 `--mode both` | **合并导出模式** | 同时导出全量实时快照、自选股概览、自选股合并历史明细及单股历史明细。 |
| `-o <路径>` 或 `--output <路径>` | **自定义输出路径** | 修改输出的 Excel 路径，默认保存至 `analysis_reports/stock_export_report.xlsx`。 |
| `-n <天数>` 或 `--offset <天数>` | **自定义历史天数** | 修改自选股历史 K 线抓取长度，默认 350 天。 |

## 三、 使用与运行指南

### 1. 查看与管理自选股
自选股可以在 GUI 客户端的“自选股管理”中添加/拖拽排序，或在配置文件中修改：
- 配置文件路径：`C:\Users\<您的用户名>\AppData\Local\stock_monitor\config.json`
- 配置项：`user_stocks` 数组（例如 `["sh000001", "sz000559", "sz002600"]`）

### 2. 运行脚本生成报表
激活项目虚拟环境并运行如下命令：

```powershell
# 1. 激活项目虚拟环境
.venv\Scripts\activate

# 2. 选择不同模式执行导出

# 模式一：仅导出自选股指标历史明细（默认）
python scripts/reporting/export_stocks_to_excel.py

# 模式二：仅导出全量 A 股实时快照
python scripts/reporting/export_stocks_to_excel.py --mode all

# 模式三：自选股历史指标 + 全量实时数据，合并导出到同一个 Excel
python scripts/reporting/export_stocks_to_excel.py --mode both

# 额外参数示例：指定导出 100 天历史数据，并输出至指定路径
python scripts/reporting/export_stocks_to_excel.py --mode watchlist --offset 100 --output D:/stocks_100d.xlsx
```
