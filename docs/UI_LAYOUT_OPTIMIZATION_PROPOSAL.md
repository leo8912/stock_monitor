# 设置界面 UI 布局优化方案

## 📊 当前 UI 问题总结

### 视觉问题
1. **自选股管理区域**
   - 搜索框和列表高度不一致，视觉不平衡
   - 添加按钮位置过低，与搜索框分离
   - 删除按钮颜色过深（红色），过于突兀
   - 列表项高度过大，浪费空间

2. **显示设置区域**
   - 字体大小、字体选择、透明度挤在一行
   - 滑块长度不一致，视觉混乱
   - "恢复默认"按钮位置不明显

3. **量化分析预警区域**
   - 标题"(附带微信机器人强推)"冗余且过长
   - 复选框标签过长，包含技术细节
   - 字段标签不统一（英文冒号 vs 中文冒号）

4. **底部设置区域**
   - "开机启动"、"刷新频率"、"版本号"挤在一起
   - 元素对齐不一致
   - "检查更新"按钮样式不明显

### 布局问题
- 三个大区域视觉分离不够
- 缺少明确的分组和分隔
- 按钮样式不统一
- 输入框长度不一致

---

## 🎨 优化方案

### 方案 1: 卡片式布局（推荐）⭐

**特点**:
- 每个区域使用独立的卡片容器
- 增加卡片标题和图标
- 统一间距和内边距
- 使用不同的背景色区分层级

**实施难度**: 中等
**视觉效果**: ⭐⭐⭐⭐⭐
**用户体验**: ⭐⭐⭐⭐⭐

### 方案 2: 分组优化布局

**特点**:
- 保持现有框架
- 优化内部元素排列
- 统一标签和输入框
- 改进按钮样式

**实施难度**: 低
**视觉效果**: ⭐⭐⭐⭐
**用户体验**: ⭐⭐⭐⭐

### 方案 3: 现代简约布局

**特点**:
- 减少视觉干扰
- 增加留白空间
- 使用图标替代文字
- 扁平化设计

**实施难度**: 高
**视觉效果**: ⭐⭐⭐⭐⭐
**用户体验**: ⭐⭐⭐⭐

## 🎨 颜色和样式规范

### 按钮颜色
```python
# 主操作按钮（确定）
primary_color = "#0078D4"  # 微软蓝

# 次操作按钮（取消）
secondary_color = "#6C757D"  # 灰色

# 危险操作按钮（删除）
danger_color = "#DC3545"  # 红色（降低饱和度）

# 功能按钮（测试、检查更新）
success_color = "#28A745"  # 绿色

# 信息按钮（使用说明）
info_color = "#17A2B8"  # 青色
```

### 间距规范
```python
# 卡片外边距
card_margin = 10px

# 卡片内边距
card_padding = 15px

# 元素间距
element_spacing = 8px

# 分组间距
group_spacing = 20px

# 按钮间距
button_spacing = 10px
```

### 字体规范
```python
# 分组标题
title_font = "微软雅黑 14px Bold"

# 标签字体
label_font = "微软雅黑 9px"

# 输入框字体
input_font = "微软雅黑 10px"

# 按钮字体
button_font = "微软雅黑 10px"
```
